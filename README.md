## How to Run

### Start the Database
Ensure MySQL is running on `localhost:3306`.

Create the database: Open your MySQL credentials (e.g., MySQL Workbench or Command Line) and run:

```sql
CREATE DATABASE todo_db;
````

Update `.env` if your credentials differ from `root:password`.

---

### Install Dependencies

```bash
cd todo_api
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

*(I have already installed the dependencies in the venv folder)*

### Run Migrations (Required)

Ensure you are in the `todo_api` directory and the virtual environment is verified (or use the full path).
